## ğŸ“ Content Management

### Adding Blog Posts

Edit `src/data/blogPosts.tsx` and add a new entry to the `blogPosts` array:

```tsx
{
  slug: 'your-post-slug',
  title: 'Your Post Title',
  date: '2024-MM-DD',
  excerpt: 'Brief description...',
  content: (
    <div className="prose prose-lg max-w-none">
      {/* Your blog post content in JSX */}
    </div>
  ),
}
```

### Updating Projects

Edit `src/pages/Projects.tsx` and modify the `projects` array.

### Updating Personal Info

- **Home page:** `src/pages/Home.tsx`
- **About Us page:** `src/pages/AboutUs.tsx`
- **PhilArchive link:** `src/pages/PhilArchive.tsx` (replace `YOUR_PHILARCHIVE_URL_HERE`)

## ğŸ¨ Customization

### Colors and Styling

The site uses Tailwind CSS. To customize:

1. Edit `tailwind.config.js` for theme extensions

1. Place images in `public/` directory
2. Reference them in your code as `/image-name.jpg`

Example for About Us page photo:
```tsx
<img src="/couple-photo.jpg" alt="Description" />
```

## ğŸ”§ Troubleshooting

### Build Errors

- Ensure all dependencies are installed: `npm install`
- Clear cache and rebuild: `rm -rf node_modules dist && npm install && npm run build`

### Routing Issues on GitHub Pages

If direct URLs don't work (404 on refresh), this is normal for single-page apps on GitHub Pages. The current setup handles this with React Router's `BrowserRouter`.

### Custom Domain Not Working

1. Verify `public/CNAME` contains your domain: `yanylin.me`
2. Check your domain's DNS settings point to GitHub Pages
3. Wait up to 24 hours for DNS propagation
